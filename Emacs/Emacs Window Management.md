# Emacs Window Management

## Frames

- `C-x 5 2`: Create a new frame
- `C-x 5 0`: Delete current frame
- `C-x 5 o`: Cycle through frames

## Windows

- `C-x 0`: Close window
- `C-x o`: Switch window
- `C-x 1`: Close other windows
- `C-x 2`: Split window vertically
- `C-x 3`: Split window horizontally
