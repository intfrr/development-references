# `vim` Editing

## Insert Mode

- `cc`: Enter insert mode at correct indent level
- `^n`: Autocomplete
- `^y`: Accept from autocomplete list

## Selecting

- `⌃v`: Visual Block Mode
- `gv`: Reselect last visual block

## Undo

- `u`: Undo
- `⌃r`: Redo

## Numbers

- `C-a` / `C-x`: Increment/decrement number
